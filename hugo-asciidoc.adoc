== Hugo Asciidoc Cheatsheet

This is a highly personlized cheatsheet I use to store shortcuts and cheats for my own asciidoc blog articles published via hugo. Shared in case it is useful.


=== Source Code Syntax Highlighting

1) Use traditional asciidoc syntax:


[source,bash]
----
export WORKON_HOME=$HOME/.virtualenvs                 <1>
export PIP_VIRTUALENV_BASE=$WORKON_HOME
export PIP_RESPECT_VIRTUALENV=true
----

2) Use shortcodes

{{< highlight bash  >}}
$ source ~/.zshrc
{{< /highlight >}}

.Hugo Commands

|===

|New post
|`hugo new posts/my-first-post.md`

|New Site      
|`hugo new site <foo>`

| Build site
| `hugo`

|Run server and build drafts
|`hugo server  --buildDrafts` or `-D`

|Run on a different port (default 1312)
|`hugo server -D --port=1314`

|===
